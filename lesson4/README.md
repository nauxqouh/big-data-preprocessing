# PySpark Basics

## Prerequisite

- Python 3.13 or higher.
- Install [uv](https://docs.astral.sh/uv/getting-started/installation/) on your system.

## Setting Up Jupyter

If it's not already, install Jupyter Notebook using uv:

```bash
uv add jupyter
```

## Installing PySpark

Install PySpark using uv:

```bash
uv add pyspark
```

**_Note:_** If you clone this project, you can run `uv sync --locked` to install all package dependencies.
